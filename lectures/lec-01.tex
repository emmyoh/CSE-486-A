\subsection{Uninformed Search}
\label{sub_sec:uninformed_search}

Many AI tasks can be formulated as search problems; the goal is to find a \emph{sequence of actions}.
\begin{itemize}
    \item Puzzles
    \item Games
    \item Navigation
    \item Assignment
    \item Motion planning
    \item Scheduling
    \item Routing
\end{itemize}
